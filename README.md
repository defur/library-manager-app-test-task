Task Overview
Build a user-friendly web application for managing a library with books and authors. The
application should allow users to perform the following operations:
1. Add a new book to the library.
2. View a list of all books.
3. Update details of an existing book.
4. Delete a book from the library.
5. Filter a list of books by title or author.
6. Add a new author.
7. Delete an author and their books
8. Export a list of books.
9. Log in and log out
Deliverables
Provide a link to a Git repository containing the source code. Include a README file with
instructions.
